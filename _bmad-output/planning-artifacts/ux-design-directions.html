The "Vault" Pivot Decision

Atmosphere

Transition from "Public Discovery" to "Private Operation." The UI should feel tighter, more focused, and strictly authoritative upon entry.

Layout

Use a sidebar-driven navigation for the workspace to maximize vertical space for data tables, while the landing page remains horizontally centered.

Density

Dashboard components use 50% less padding than Landing Page equivalents to allow for high-volume portfolio management (up to 100 partners).
